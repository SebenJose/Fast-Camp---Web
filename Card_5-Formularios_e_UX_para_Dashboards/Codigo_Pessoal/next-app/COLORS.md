# Paleta de Cores - Dashboard

## Cores do Projeto

| Nome       | Hex     | Uso                      |
| ---------- | ------- | ------------------------ |
| Vermelho   | #E43636 | CTAs, Destaques, Ações   |
| Bege Claro | #F6EFD2 | Background secundário    |
| Bege Médio | #E2DDB4 | Bordas, Elementos subtis |
| Preto      | #000000 | Texto principal          |

## Referência Rápida

```
Vermelho:      #E43636
Bege Claro:    #F6EFD2
Bege Médio:    #E2DDB4
Preto:         #000000
```

## Como Usar

### Tailwind CSS

```jsx
<button className="bg-red text-white">CTA</button>
<div className="bg-cream-light">Content</div>
<p className="text-black">Texto principal</p>
```
